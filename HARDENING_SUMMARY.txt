================================================================================
HOW TO USE
================================================================================

1. Install the watchdog to run at startup:
   $ cd /mnt/crypto_trading_agent
   $ ./install_watchdog.sh

2. Verify installation:
   $ crontab -l | grep watchdog.sh

3. Monitor watchdog logs:
   $ tail -f /mnt/crypto_trading_agent/logs/watchdog.log

4. Manual test (optional):
   $ /mnt/crypto_trading_agent/watchdog.sh

Expected output in logs/watchdog.log:
   [2026-03-22 16:30:01] === Watchdog cycle started ===
   [2026-03-22 16:30:01] OK: state.json is fresh (45s old)
   [2026-03-22 16:30:02] OK: Agent process and state are healthy
   [2026-03-22 16:30:02] === Watchdog cycle completed ===
